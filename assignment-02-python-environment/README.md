# Assignment 2: Python Environment Setup

## About This Exercise

This was a guided university laboratory exercise about preparing a Raspberry Pi or Jetson Nano for Python machine-learning work.

## What I Did

- Downloaded an ARM-compatible Anaconda installer
- Transferred the installer to the Raspberry Pi/Jetson Nano
- Installed Anaconda using the Linux terminal
- Installed the scikit-learn Python package
- Opened Python and checked that scikit-learn could be imported

## What These Tools Are

### Anaconda

Anaconda is a tool used to install Python and manage Python packages and environments.

### scikit-learn

scikit-learn is a Python library containing common machine-learning algorithms.

### ARM64

ARM64 is a processor architecture used by devices such as Raspberry Pi and Jetson Nano.

## What I Learned

This exercise taught me that software must be installed in a version that matches the computer's operating system and processor architecture.

It also introduced me to installing Python libraries and testing whether they were installed correctly.
