# Assignment 1: Linux and Git Basics

## About This Exercise

This was a guided university laboratory exercise introducing basic Linux terminal commands, Bash scripting and Git.

## What I Practised

* Navigating between folders using the terminal
* Creating, copying, moving and deleting files
* Creating and compressing directories
* Changing file permissions
* Writing and running a basic Bash script
* Starting and stopping a background process
* Initialising a Git repository
* Adding and committing a file with Git
* Learning the basic syntax of the `scp` command

## What I Learned

This exercise introduced me to controlling a Linux computer through typed commands rather than a graphical interface.

It also introduced Git, which is used to record and manage changes made to project files.
